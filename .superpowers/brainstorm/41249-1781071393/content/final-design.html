# 决策 方案 援引来源
1 技术栈 React 19 + R3F v9 + TypeScript pmndrs 生态 45k ★，TresJS 4k ★
2 滚动驱动 GSAP ScrollTrigger（命令式）+ R3F（声明式渲染） Codrops 2025, Builder.io, Ringston 3D
3 状态管理 Zustand（渲染状态）+ React useState（UI 状态） R3F 官方"五类出口", Galaxy Voyager, HekTek City
4 Act 可见性 visible prop 切换（始终挂载） R3F Performance Pitfalls: "Use visibility instead"
5 粒子系统 3 主行星 = 独立 Mesh, 132 碎片 = InstancedMesh Drei Sparkles, Galaxy Voyager
6 测试策略 L1 纯函数全覆盖 + L2 关键路径场景图验证 R3F Testing docs, @react-three/test-renderer
