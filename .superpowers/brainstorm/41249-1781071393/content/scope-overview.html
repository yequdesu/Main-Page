重构范围：Vue + Vanilla Three.js → React + R3F + TypeScript

当前架构 vs 目标架构 — 哪些保留、哪些重写、哪些新增

当前架构（Before）

Vue 3 + Vanilla Three.js + GSAP · 14 个源文件
┌─ main.js               Vue 入口 + GSAP 注册
├─ App.vue               ← 滚动物理、DOM 叠加层、桥接 Three.js
├─ LighthouseScene.vue   ← 🎯 1580行，全部 Three.js 逻辑
│   ├── Act 1 (727行)    海洋/灯塔/光束/粒子
│   ├── Act 2 (127行)    网格过渡
│   └── Act 3 (303行)    轨道/恒星/标签/聚焦
├─ AppFooter.vue         静态页脚
├─ main.css              全局样式
├─ vite.config.js
├─ stats_server.py       ← Python 后端（独立）
├─ mainpage.nginx.*.conf ← Nginx 配置（不变）
└─ package.json           Vue + GSAP + Three.js

目标架构（After）

React 19 + R3F v9 + GSAP + TypeScript · ~25 个源文件
┌─ src/main.tsx               React 入口 + GSAP 注册
├─ src/App.tsx                ← 滚动物理、DOM 叠加层（保留逻辑）
├─ src/r3f/
│   ├── Canvas.tsx            ← R3F Canvas 配置
│   ├── ScrollRig.ts          ← 📦 集中阈值定义
│   └── hooks/
│       ├── useScrollProgress.ts   ← scroll → R3F 桥接
│       └── useFrameCache.ts       ← 帧缓存守卫
├─ src/acts/                  ← 🆕 每个 Act 是独立组件
│   ├── Act1OceanVoyage.tsx
│   ├── Act2GridTransition.tsx
│   └── Act3ContentPhase.tsx
├─ src/actors/                ← 🆕 3D 实体（取代 build* 函数）
│   ├── Lighthouse.tsx
│   ├── LightBeam.tsx
│   ├── OceanWaves.tsx
│   ├── DustField.tsx
│   ├── CentralStar.tsx
│   ├── OrbitRings.tsx
│   └── PlanetLabel.tsx
├─ src/behaviors/             ← 🆕 可组合行为 hook
│   ├── useScreenSpaceHover.ts
│   ├── useOrbitPosition.ts
│   ├── useAppearanceFade.ts
│   └── useOcclusionFade.ts
├─ src/shaders/
│   └── VolumetricBeamShader.ts
├─ stats_server.py            ← 不变
├─ mainpage.nginx.*.conf      ← 不变
└── vitest + @react-three/test-renderer ← 🆕 测试基础设施

迁移清单

组件 操作 说明
Vue 3 ⬤ 移除 完全替换为 React 19
LighthouseScene.vue ⬤ 移除 拆分为 15+ 个 R3F 组件和 hook
App.vue ⬤ 重写 逻辑保持，框架从 Vue 改为 React
GSAP + ScrollTrigger ⬤ 保留 通过 @gsap/react useGSAP 集成
物理滚动逻辑 ⬤ 保留 转换为 React hook/ref 模式
Three.js 几何体/材质 ⬤ 重构 从命令式 → 声明式 JSX
stats_server.py ⬤ 不变 独立 Python 后端
Nginx 配置 ⬤ 不变 部署方式不变
预分配对象池 ⬤ 保留模式 转换为 R3F 的 ref 直接操作模式
帧缓存守卫 ⬤ 保留 + 改进 抽取为 useFrameCache hook

不变的部分

完全保留

  • 所有视觉效果（波浪形状、光束颜色、粒子行为）—— 数学逻辑不变，只改组织方式
  • GSAP ScrollTrigger + 物理滚动参数（FRICTION、MAX_VELOCITY）
  • stats_server.py + Nginx 配置
  • 品牌文字样式、聚焦 SVG 叠加层动画

完全重写

  • Vue → React：所有 .vue 文件 → .tsx 文件
  • 命令式 Three.js → 声明式 R3F JSX
  • Act 管理 → React conditional rendering
  • 模块级变量 → Zustand 瞬态 + R3F ref
  • JavaScript → TypeScript
